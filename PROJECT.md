# 🐝 ВУЛИК (VULYK) — Екосистема для пасічників і поціновувачів меду

> **Одне слово, одне місце, одна спільнота.** Super-app, де зустрічаються пасічники, покупці, постачальники обладнання та фермери — від вулика до банки і далі до столу.

---

## 📌 Виконавче резюме

**ВУЛИК** — це гібридна платформа (mobile + web), що поєднує:

1. **Інструмент управління пасікою** (журнал оглядів, лікування, медозбір)
2. **Маркетплейс** (мед, обладнання, бджолопакети, інвентар — нове + б/в)
3. **Соцмережа / форум** з регіональними гілками
4. **Логістика кочівлі** (карта медоносів, домовленості з фермерами, маршрути)
5. **Цифровий QR-паспорт меду** з верифікацією походження
6. **Програма "Усинови вулик"** для віральної залученості міських покупців

Платформа вирішує **6 системних болей** галузі одночасно, замість того щоб бути ще одним вузькоспеціалізованим додатком (яких уже десятки і всі — на одну функцію).

**Цільові ринки:** Україна (старт) → Польща, Молдова, Румунія, Туреччина, Грузія → ЄС.

**Стартова монетизація:** комісія маркетплейсу + freemium підписка для пасічників. Потенціал ARR в Україні: $1.5–3M за 24 місяці (при 5% проникненні в 400k активних пасічників).

## 🔍 Аналіз ринку та виявлені болі

### Масштаб ринку
- **Україна:** ~400 000 пасічників (один із найбільших ринків у Європі), 70–80 тис. тонн меду на рік, експортер №1 у ЄС.
- **Світ:** глобальний ринок меду ~$10B, зростає 5–6% на рік.
- **Глобально** ~90M вуликів, ~10M пасічників-аматорів.

### Системні болі (з форумів, статей, відгуків)

| # | Біль | Кого зачіпає | Що це коштує |
|---|------|--------------|--------------|
| 1 | **Перекупники диктують ціну** (120–140 грн/л при собівартості вищій) | Пасічники | Втрата 30–50% маржі |
| 2 | **Покупці не довіряють меду** (14% меду в ЄС — фальсифікат, мед — 3-й найбільш підроблюваний продукт) | Споживачі | Купують у супермаркеті замість прямо в пасічника |
| 3 | **75% пасічників кидають перший застосунок** через брак фіч | Пасічники-цифрові | Розпорошеність, відсутність єдиного джерела |
| 4 | **Логістика кочівлі** — пошук медоносних полів, домовленості з аграріями, координація з іншими пасічниками (не наїхати один на одного) | Промислові пасічники | Втрати медозбору, конфлікти |
| 5 | **Втрата роїв** через брак моніторингу між оглядами | Усі пасічники | Втрата до 30% сім'ї |
| 6 | **Розпорошений ринок Б/У обладнання** — медогонки, рамки, вулики продають через OLX/Facebook без верифікації | Усі | Шахрайство, незручність |
| 7 | **Молодим пасічникам нікуди звернутись** за наставництвом, окрім хаотичних чатів | Початківці | Висока смертність бджіл на старті |
| 8 | **Облік для держави та ЄС** (ветеринарна звітність, реєстр пасік) — заповнюється вручну | Усі офіційні | Час, штрафи |

### Існуючі рішення та чому вони не закривають проблему
- **HiveTracks, ApiaryBook, BeeAttended, ApiNote, Beekeeping Diary** — лише журнал оглядів. Немає маркетплейсу, спільноти, верифікації меду.
- **Honeytrail, FoodTraze** — лише трейсабіліті, дорого, B2B, не для української маси.
- **OLX / Prom / Facebook Marketplace** — немає фокусу на меді, немає верифікації, немає інструментів для пасічників.
- **Pasika.news, форум ТочкУ** — контент-сайти, без інтерактивних інструментів.
- **Apis Cooperative** — кооперативи існують офлайн, але без зручного цифрового інструменту.

**Висновок:** ринок фрагментований. Ніхто не закриває всі болі в одному продукті. Це — основна можливість.

## 🎯 Концепція продукту

### Філософія
> "Як Strava для бігунів, як Untappd для пива, як Komoot для туристів — **ВУЛИК для пасічників**."

Один застосунок, де:
- **Пасічник** веде свою пасіку, продає мед, купує обладнання, спілкується.
- **Покупець** знаходить чесний мед поруч, читає історію пасіки, підписується на доставку.
- **Постачальник обладнання** має канал збуту з низькою комісією.
- **Аграрій** знаходить пасічників для запилення полів за гроші.

### Ключовий інсайт
**Двосторонність — це віральний двигун.** Кожен покупець, який сканує QR на банці меду, стає потенційним користувачем платформи → бачить інших пасічників → ділиться → запрошує. Кожен пасічник, який зареєструвався, тягне за собою своїх клієнтів.

## 🧩 Модулі продукту

### MVP (місяці 1–4)

#### 1. 📔 Журнал пасіки (для пасічника)
- Картка кожного вулика: матка (рік, лінія), сила сім'ї, корм, лікування
- **Голосовий ввід оглядів** (як ApiNote, але українською) — щоб руки в рукавицях
- Офлайн-режим (на пасіці немає інтернету) → синхронізація
- Шаблони оглядів: весняна ревізія, контроль роїння, осіння підготовка
- **Розумні нагадування:** "Через 21 день перевірити нову матку у вулику №7"
- Експорт у PDF/CSV для ветеринарної звітності
- Інтеграція з державним реєстром пасік (через API, коли стане доступним)

#### 2. 🛒 Маркетплейс
- **3 категорії:**
  - **Мед** (липа, акація, гречка, соняшник, різнотрав'я, експрес-доставка)
  - **Обладнання** нове + б/у (медогонки, рамки, вулики, костюми, віск, вощина)
  - **Бджолосім'ї / пакети / матки** (з гарантією, верифіковані продавці)
- **Геофільтр**: "у радіусі 50 км", "у моїй області", "по всій Україні з доставкою"
- **Внутрішні платежі** з ескроу (гроші заморожені, поки покупець не отримав)
- **Низька комісія**: 3–5% (проти 10–15% у Prom/Etsy)
- **Логістика** через інтеграцію з Новою Поштою / Укрпошта / Meest

#### 3. 🗺️ Карта пасічників і медоносів
- Кожна верифікована пасіка на мапі (з дозволу власника)
- **Шар медоносів:** липа, акація, соняшник, гречка — фази цвітіння за регіонами
- Інтеграція з NASA FIRMS / Sentinel: моніторинг обробок полів пестицидами (виживання бджіл!)
- Покупцеві: "знайди пасіку біля мене → купи мед напряму"

#### 4. 🏅 QR-паспорт меду
- Кожна банка / партія отримує **унікальний QR**
- Сканування → сторінка з:
  - Фото пасічника + його профіль і відгуки
  - Регіон, GPS-координата пасіки (з точністю до села)
  - Сорт меду, рік відкачки, лабораторні аналізи (опц.)
  - Відео з пасіки (TikTok-стиль, 15 сек)
- **Цей екран сам по собі — лендинг застосунку:** "Підпишись на цього пасічника", "Знайди ще 12 пасік поруч"

#### 5. 💬 Спільнота
- Регіональні гілки форуму (Вінниччина, Закарпаття, Полтавщина...)
- Q&A з тегами (хвороби, матки, обладнання)
- Лайв-чати ("новачок → ментор")
- Стрічка історій з пасік (як Instagram Stories)

### Фаза 2 (місяці 5–9)

#### 6. 🛣️ Кочівля 2.0
- **"Аграрна біржа"**: фермер виставляє поле (100 га соняшника, потрібне запилення) → пасічники бідують за місце або платять за стоянку
- **Координація з іншими пасічниками**: бачиш, хто вже стоїть у радіусі 3 км
- Прогноз погоди + цвітіння → найкращий день виїзду

#### 7. 🤝 Кооперативний продаж
- Пасічники одного регіону формують **спільний лот** ("200 тонн липи від кооперативу Полтавщина")
- Платформа знаходить оптового покупця / експортера
- Розподіл оплати автоматичний
- Реальний кейс із Миколаївщини: 100 тонн через об'єднання → ціна вдвічі вища

#### 8. 🎁 "Усинови вулик" (Adopt-a-Hive)
- Міська людина платить $50–200/рік → "усиновлює" вулик
- Отримує: іменну табличку на вулику, фото/відео щомісяця, мед з "її" вулика
- Працює як **подарунок** (день народження, корпоративний подарунок екологічно-свідомим компаніям)
- **Віральне:** "Я усиновив вулик, а ти?" — share-екран з твоїм вуликом

#### 9. 📦 Підписка на мед "Honey Box"
- $15–30/місяць → банка свіжого меду з ротацією пасічників за регіоном
- "Бренд місяця" — фокус на одному пасічнику
- Як HelloFresh, але для меду + бі-продукти (прополіс, перга, пилок, віск-косметика)

### Фаза 3 (місяці 10–18)

#### 10. 🔬 Лабораторна верифікація
- Партнерство з лабораторіями (ННЦ "Інститут бджільництва ім. Прокоповича", IFC)
- Пасічник за $20 отримує "verified honey" значок на 12 місяців
- Покупець довіряє → платить премію

#### 11. 📡 IoT (опціонально)
- Інтеграція з BroodMinder, HiveMind та локальними виробниками вуликових ваг
- Пасічник бачить вагу, температуру, вібрацію → попередження про роїння

#### 12. 🎓 Академія
- Платні курси від топ-пасічників ("Як я з 10 вуликів зробив 100")
- Сертифікація → бейдж у профілі

#### 13. 🌍 Експортний хаб
- B2B-розділ: європейські оптовики → українські кооперативи
- Документообіг (фітосанітарка, сертифікати, інкотермс)

## 🌍 Регіональна адаптація

### Підхід "Регіон як перший клас"

Інтерфейс не "країна" → одразу **регіон/область**. Це принципово, бо:
- Цвітіння липи у Києві ≠ Закарпаття на 2 тижні
- Ціни на мед різняться 30% між Поліссям і Степом
- Спільноти живуть локально

### Багаторівнева геолокація
1. **Країна** (мова, валюта, регуляції)
2. **Область / воєводство / повіт** (форум, ціновий бенчмарк)
3. **Радіус навколо мене** (для маркетплейсу)

### Локалізація
- Старт: 🇺🇦 укр. + 🇬🇧 англ.
- Фаза 2: 🇵🇱 пол., 🇷🇴 рум., 🇲🇩 рум.-мд., 🇹🇷 тур., 🇬🇪 грузин.
- Фаза 3: 🇫🇷 фр., 🇩🇪 нім., 🇪🇸 ісп.

### Регіональні особливості
- **Україна:** ветеринарна звітність, реєстр пасік, експорт у ЄС
- **Польща:** AGRO Hub, дотації, інтеграція з PZP (Polski Związek Pszczelarski)
- **Туреччина:** окремі сорти (сосновий, каштановий мед — преміум)
- **Грузія:** племінне бджільництво, кавказька порода

### Цінова модель для пасічника
- **Free:** до 3 вуликів, базовий журнал, перегляд маркетплейсу, продаж до 5 оголошень/міс
- **PRO** $5/міс: безлім вуликів, аналітика, кочівля, верифікація, експорт звітності
- **Кооператив** $25/міс на 10 учасників: спільний лот, документообіг

### Юніт-економіка (грубо)
- **CAC:** $3–8 (через органіку QR-кодів + регіональні фейсбук-групи)
- **LTV пасічника PRO:** $60 (12 міс × $5)
- **LTV покупця HoneyBox:** $180/рік × 3 роки × 30% маржа = $162
- **Payback period:** < 6 місяців

## 🚀 Віральні механіки

### 1. **QR-код на банці = безкоштовний користувач**
Кожна банка меду, продана через ВУЛИК, має QR. Покупець сканує → потрапляє на лендінг → бачить інших пасічників регіону → реєструється.
**KPI:** 1 проданий мед = 2.5 нових візити.

### 2. **"Усинови вулик" як подарунок**
Корпоративні замовлення (HR-подарунки, день народження) — кожен новий "усиновитель" отримує сертифікат, яким ділиться в соцмережах.

### 3. **TikTok-стрічка з пасіки**
Пасічники знімають короткі відео (огляд вулика, відкачка). Алгоритм просуває найкращі. Зовнішній share → лідогенерація.

### 4. **Гейміфікація**
- Бейджі: "100 кг продано", "Verified", "Перший рій", "Кочівля 2026"
- Регіональні таблиці лідерів ("Топ-10 пасічників Полтавщини")

### 5. **Реферальна програма**
- Пасічник запросив пасічника → 1 міс PRO безкоштовно обом
- Покупець запросив друга → 100 грн на покупку меду

### 6. **Контент-маркетинг**
- YouTube/TikTok серіал "Один день з пасічником"
- Гайди для початківців → SEO трафік
- Колаборації з ютуберами-пасічниками (Сильна Пасіка, ПасікаПро)

### 7. **Сезонні івенти**
- "Свято першої липи" (червень) — фестиваль на платформі
- "Чорна п'ятниця обладнання" (січень — підготовка до сезону)
- "Медовий Спас" (серпень) — пік продажів

### 8. **PR-привід "Прозорість"**
- Регулярна публікація даних: середня ціна меду по областях, прогноз сезону
- Стати "Bloomberg для меду" — медіа цитуватимуть.

## 👥 Цільова аудиторія

### Пасічники
- **Аматори** (до 10 вуликів): ~70% ринку. Біль — час, простота.
- **Напівкомерційні** (10–50 вуликів): ~25%. Біль — продажі, обладнання.
- **Промислові** (50+): ~5%. Біль — логістика, експорт, кооперативи.

### Покупці меду
- **Urban Millennial / Gen Z** (25–40 років): шукає чесний продукт, готовий платити більше.
- **Сім'ї з дітьми**: безпека, локальність.
- **Здоров'я-орієнтовані**: спортсмени, ЗОЖ, веганоподібні (мед — суперфуд).
- **Корпорації** (HR-подарунки, ECO-ініціативи).

### Постачальники обладнання
- Малі виробники вуликів і рамок (зараз продають через OLX).
- Імпортери костюмів, медогонок.
- Виробники вощини, інструментів.

### Аграрії
- Власники полів соняшника / гречки / ріпаку, що потребують запилення.

## 🛠️ Технічна архітектура (висока)

### Стек (пропозиція)
- **Frontend Web:** Next.js + TypeScript + Tailwind
- **Mobile:** React Native (iOS + Android з одного коду) + offline-first (RxDB / WatermelonDB)
- **Backend:** Node.js (NestJS) або Go, PostgreSQL + PostGIS (геодані), Redis
- **Файли/медіа:** S3-compatible (Cloudflare R2 — дешево)
- **Платежі:** LiqPay / Fondy / Stripe (для EU), внутрішнє ескроу
- **Картографія:** MapLibre + OpenStreetMap, шари з NASA FIRMS, Sentinel-Hub
- **Search:** Meilisearch (швидкий, відкритий)
- **Чат:** Stream Chat або власний на WebSocket
- **Push:** Firebase Cloud Messaging
- **AI:** GPT для перетворення голосу → структурованого огляду, для модерації, для аналітики
- **Аналітика:** PostHog (self-hosted, GDPR-friendly)

### Інфра
- **Хмара:** Hetzner / OVH (дешеві в ЄС) → AWS, коли вийдемо в США
- **CI/CD:** GitHub Actions
- **Моніторинг:** Sentry, Grafana

### Безпека / комплаєнс
- GDPR (важливо для ЄС)
- Закон України "Про захист персональних даних"
- PCI DSS для платежів (через прошарок)

**Головна метрика:** **літрів меду продано через платформу/місяць.**

## 🎬 Чому це спрацює

1. **Реальні болі, не вигадані.** Кожен модуль — відповідь на конкретну скаргу з форумів.
2. **Двосторонній мережевий ефект.** Кожен покупець підсилює пасічника, кожен пасічник — платформу.
3. **Природний вірал** через QR на банках — не потрібно платити за більшість трафіку.
4. **Низька конкуренція в Україні** і всій Східній Європі — лише вузькі застосунки.
5. **Висока маржа маркетплейсу** + рекурентний дохід підписок.
6. **Сильний emotional hook:** бджоли — це любов, екологія, родинність. Працює на media і інвесторів.
7. **Експортна перспектива:** з одного MVP легко вийти в 12 країн.

## 🤝 Що далі

1. **Валідація:** провести 20 коротких інтерв'ю з пасічниками різних рівнів — підтвердити пріоритети болей.
2. **Прототип-демо:** Figma-кліклабельний прототип ключових екранів (журнал, маркетплейс, QR-сторінка).
3. **Landing page:** збір ранніх реєстрацій + waitlist (домен `vulyk.app` чи `medarnik` як локалізована назва).
4. **Команда:** 1 продакт, 1 фул-стек, 1 мобайл, 1 дизайнер, 1 community-менеджер з пасічного фону.
5. **Раунд seed:** $150–300K (12 місяців ranway до публічного запуску).
